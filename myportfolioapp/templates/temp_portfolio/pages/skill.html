Firebase
A service from Google to provide convenience and even make it easier for application developers to develop their applications. Firebase's first product was Realtime Database. Realtime Database is used by developers to store data and synchronize it to many users. Then it developed as an application developer service.

RxDart
One of the popular libraries used in reactive programming in Flutter. This library allows you to use Rx in your Flutter code easily. RxDart provides StreamBuilder which makes it easy to build reactive Flutter apps. StreamBuilder can update widgets when changes occur in the data stream.
BloC
An architectural pattern that separates business logic and state management from the UI, making applications easier to maintain, test, and evolve. BLoC allows the UI to receive state changes from the BLoC without knowing how the changes occurred, and allows the UI to send events (actions) to the BLoC.
